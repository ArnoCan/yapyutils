Module Hierarchy

  • yapyutils: yapyutils provides various low-level Python utilities for installation and bootstrap stages, thus the implementation is kept independent from higher stack-layers when ever possible. Ultimately features are provided redundant in favor of independence.
    • yapyutils.modules: yapyutils.modules provides various low-level Python utilities for modules.
      • yapyutils.modules.loader: yapyutils.modules.loader provides module location and load as file.
  • yapyutils.help: yapyutils.help provides various low-level Python utilities for runtime help information.
  • yapyutils.releases: yapyutils.releases provides various low-level Python utilities for version and release numbering.
  • yapyutils.config: yapyutils.config provides various low-level Python utilities for configuration files and configuration setups.
    • yapyutils.config.capabilities: yapyutils.config.capabilities provides simple hierarchical configuration file utils based on in-memory JSON representation. This includes nested and chained configuration data branches.
  • yapyutils.files: yapyutils.files provides various low-level Python utilities for file systems and files.
    • yapyutils.files.finder: yapyutils.files.finder provides file locations.
    • yapyutils.files.utilities: yapyutils.files.utilities provides file helpers.
